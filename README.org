* Org Roam Ui
  Based on and builds on the code from [[https://github.com/org-roam/org-roam-ui][here]].

** Installation
    Requires you to have =node= and =websocat= install.

    Assumuing you are using lazy:
    #+begin_src lua
    {
        "gingerfocus/org-roam-ui",
        build = "npm install",
        dependencies = {
            "nvim-lua/plenary.nvim",
            -- assuming this is configured elsewhere
            "chipsenkbeil/org-roam.nvim",
        },
        opts = {},
    }
    #+end_src

    Then in your editor run =:OrgRoamUiOpen=.
